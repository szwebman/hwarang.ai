# AI Hub 한국어 데이터 활용 가이드

## AI Hub란?

[AI Hub](https://aihub.or.kr)는 **한국지능정보사회진흥원(NIA)** 이 운영하는 국가 AI 데이터 플랫폼입니다.
한국 정부 예산으로 구축된 **고품질 한국어 데이터셋**을 무료로 제공합니다.

> **HuggingFace에 없는** 한국어 특화 데이터의 보고입니다.

## 가입 + 신청 방법

### 1. 회원가입 (필수)
1. https://aihub.or.kr 접속
2. 회원가입 (개인 또는 법인 - 퍼시스모어 법인 추천)
3. 본인 인증 (공동인증서 또는 휴대폰)

### 2. 데이터 신청
1. 원하는 데이터 검색
2. "데이터 다운로드 신청" 클릭
3. 사용 목적 작성 (예: "한국어 LLM 학습 연구")
4. 일반적으로 즉시 승인 (일부는 1~3일 검토)

### 3. 다운로드
- 승인 후 마이페이지에서 다운로드
- 대용량은 분할 ZIP으로 제공
- 일부는 FTP로 받음

## 우선 다운로드 추천 순서

### 1순위 (꼭 받기) ⭐⭐⭐

```
1. 한국어-영어 번역(병렬) 말뭉치
   → 한국어-영어 양방향 학습에 필수
   → 코드 + 자연어 페어 효과

2. 법률 문서 코퍼스 + 법률 QA 데이터
   → Legal 도메인 핵심
   → 다른 곳에서 구하기 어려움

3. 회계·세무 문서 + 금융 도메인
   → Tax 도메인 핵심

4. 소프트웨어 개발 말뭉치
   → Code 도메인 한국 특화
```

### 2순위 (좋은 보강) ⭐⭐

```
5. 한국어 SNS 말뭉치 → 구어체
6. 방송 콘텐츠 → 자연스러운 한국어
7. KorQuAD 2.0 → MRC 능력
8. 한국어 대화 코퍼스 → 대화 능력
```

### 3순위 (선택) ⭐

```
9. 행정 문서 → 공문 스타일
10. 한국어 강의 → 학술 스타일
11. 상식 추론 → 한국 문화 이해
```

## 변환 스크립트 (AI Hub → Hwarang 형식)

각 데이터셋을 Hwarang 학습 형식으로 변환하는 스크립트를 제공합니다.

### 예: 법률 QA → SFT JSONL

```python
# scripts/data/convert_aihub.py
import json
from pathlib import Path

def convert_legal_qa(aihub_dir: Path, output: Path):
    """AI Hub 법률 QA → SFT JSONL"""
    examples = []

    for json_file in aihub_dir.glob("*.json"):
        with open(json_file, encoding="utf-8") as f:
            data = json.load(f)

        for item in data.get("data", []):
            question = item.get("question", "").strip()
            answer = item.get("answer", "").strip()

            if question and answer:
                examples.append({
                    "messages": [
                        {"role": "user", "content": question},
                        {"role": "assistant", "content": answer},
                    ],
                    "domain": "legal",
                    "source": "aihub",
                })

    with open(output, "w", encoding="utf-8") as f:
        for ex in examples:
            f.write(json.dumps(ex, ensure_ascii=False) + "\n")

    print(f"변환 완료: {len(examples)}개 → {output}")
```

## 라이선스 주의사항

AI Hub 데이터는 **무료지만 라이선스가 다양**합니다:

| 라이선스 | 설명 | 상업 이용 |
|---------|------|----------|
| **CC BY** | 출처 표시만 | ✅ 가능 |
| **CC BY-NC** | 비상업적만 | ⚠️ Hwarang 상용화 시 주의 |
| **CC BY-SA** | 동일 라이선스 | ✅ 가능 (조건 있음) |
| **공공누리 1유형** | 출처 표시 | ✅ 가능 |
| **공공누리 4유형** | 출처+비상업+변경금지 | ❌ 학습 데이터로 부적합 |
| **연구 목적 한정** | 연구용만 | ❌ 상용 불가 |

> ⚠️ **반드시 각 데이터셋의 라이선스를 확인하고**, 상업 이용 가능한 것만 학습에 포함하세요.
> 비상업 라이선스는 연구 단계에서만 사용.

## 자동화 한계

AI Hub는 **API가 없습니다.** 따라서:
- 자동 다운로드 ❌ (수동 신청 + 다운로드)
- 일괄 처리 ❌ (각 데이터셋 개별 신청)
- 빠른 수집 ❌ (승인 1~3일 소요)

대신 **품질이 압도적으로 좋고**, **한국 특화 데이터**가 많아서 시간 들일 가치가 있습니다.

## 추천 워크플로우

```
Week 1: AI Hub 가입 + 1순위 데이터 신청
Week 2: 1순위 데이터 다운로드 + 정리
Week 3: 2순위 신청 + 1순위 변환 스크립트
Week 4: 2순위 다운로드 + Hwarang 학습 데이터로 통합
```

이렇게 하면 **공개 HuggingFace 데이터 + AI Hub 한국 특화 데이터**가 결합되어
한국어 LLM 중에서도 차별화된 데이터 품질을 확보할 수 있습니다.
