# 화랑 Grid Agent — 5분 빠른 시작

> 더 자세한 내용은 [USER_GUIDE.md](./USER_GUIDE.md) 를 참고하세요.

---

## 1단계 — 다운로드 + 설치 (1분)

| 내 OS | 다운로드 | 설치 방법 |
|---|---|---|
| **macOS** | [.dmg](https://hwarang.ai/download/mac) | 더블클릭 → Applications 로 드래그 |
| **Windows** | [.msi](https://hwarang.ai/download/win) | 더블클릭 → "다음다음" |
| **Linux** | [.deb](https://hwarang.ai/download/deb) / [.AppImage](https://hwarang.ai/download/linux) | `sudo dpkg -i ...` 또는 `chmod +x` 후 실행 |

---

## 2단계 — 로그인 (1분)

1. 설치된 **화랑 Grid Agent** 실행
2. 트레이/메뉴바 아이콘 클릭 → 창 열기
3. **이메일 + 비밀번호** 로 로그인 (회원가입은 1분이면 됩니다)

---

## 3단계 — KYC 본인 인증 (3분)

> **출금 시에만** 필요합니다. 적립은 KYC 없이도 됩니다.

1. 대시보드 → **본인 인증** 클릭
2. **신분증 사진** + **셀카 1장** 업로드
3. 1~2일 안에 자동 승인

---

## 4단계 — 자동 시작 (0분)

설치 후 **자동으로 시작**됩니다. 아래 상태가 보이면 작업 중입니다:

| 아이콘 색 | 의미 |
|---|---|
| 🟢 **초록** | 작업 중 (HWR 적립 중!) |
| 🟡 **노랑** | 대기 중 (작업 할당 대기) |
| 🔴 **빨강** | 일시정지 (게임 / 사용자 활동 감지) |
| ⚫ **회색** | 미로그인 |

---

## 끝!

이제 컴퓨터를 켜놓기만 하면 자동으로 HWR 코인이 쌓입니다.

대시보드에서 **실시간 수익**을 확인하세요.
