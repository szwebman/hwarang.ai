## 1. 설치하기

### macOS (맥북, 아이맥)

> 대상: macOS 10.15 (Catalina) 이상

1. https://hwarang.ai/download 접속
2. **"macOS 다운로드"** 버튼 클릭
3. `Hwarang-Grid-Agent-x.x.x-universal.dmg` 파일이 다운로드됩니다
4. **다운로드 폴더**에서 `.dmg` 파일을 **더블클릭**
5. 화면에 다음과 같은 창이 뜹니다:

   ```
   ┌────────────────────────────┐
   │                            │
   │  [앱 아이콘]  →  [폴더]    │
   │  Hwarang        Applications│
   │                            │
   └────────────────────────────┘
   ```

6. **앱 아이콘**을 **Applications 폴더**로 **드래그**합니다 (왼쪽에서 오른쪽으로 끌어다 놓기)
7. 끝났습니다! `Launchpad`(우주선 모양 아이콘) 또는 `Applications` 폴더에서 **화랑 Grid Agent**를 찾아 실행하세요

> **알림**: 첫 실행 시 "확인되지 않은 개발자" 경고가 나오면, **우클릭 → 열기 → 열기**를 누르면 됩니다. (코드 사이닝이 적용된 정식 버전은 이 경고가 없습니다.)

> *(스크린샷 자리)* — 다운로드 페이지 / DMG 창 / 앱 실행 화면

### Windows (윈도우 PC)

> 대상: Windows 10 64bit / Windows 11

1. https://hwarang.ai/download 접속
2. **"Windows 다운로드"** 버튼 클릭
3. `Hwarang-Grid-Agent-x.x.x-x64.msi` 파일이 다운로드됩니다
4. 다운로드 폴더에서 **더블클릭**
5. 설치 마법사가 뜹니다:
   - **다음** 클릭
   - **사용권 동의** 체크 → **다음**
   - **설치 경로** 확인 (기본값 그대로 OK) → **다음**
   - **자동 시작** / **데스크탑 바로가기** 체크박스 (원하시는 대로) → **다음**
   - **설치** 클릭
6. 끝나면 **마침** 클릭, **"지금 실행"** 체크 후 종료하면 바로 시작됩니다

> **알림**: "Windows에서 PC를 보호했습니다" 경고가 나오면 **추가 정보 → 실행** 을 누르면 됩니다.

> *(스크린샷 자리)* — 다운로드 / 설치 마법사 단계별

### Linux (우분투, 데비안 등)

#### Ubuntu / Debian (.deb)

```bash
# 1. 다운로드
wget https://hwarang.ai/download/hwarang-grid-agent_0.1.0_amd64.deb

# 2. 설치
sudo dpkg -i hwarang-grid-agent_0.1.0_amd64.deb
sudo apt-get install -f   # 의존성 자동 해결

# 3. 실행
hwarang-grid
```

#### 모든 배포판 (AppImage)

```bash
# 1. 다운로드
wget https://hwarang.ai/download/Hwarang-Grid-Agent-0.1.0-x86_64.AppImage

# 2. 실행 권한
chmod +x Hwarang-Grid-Agent-0.1.0-x86_64.AppImage

# 3. 실행 (더블클릭 또는)
./Hwarang-Grid-Agent-0.1.0-x86_64.AppImage
```

## 2. 첫 실행

### 트레이 아이콘 찾기

설치 후 화랑 아이콘이 **항상 켜져 있는 상태**로 동작합니다.

| OS | 위치 |
|---|---|
| macOS | 화면 **상단 메뉴바** (시계 옆) |
| Windows | 화면 **우측 하단 트레이** (시계 왼쪽). 안 보이면 **^** 버튼 클릭 |
| Linux | 트레이 아이콘 영역 (DE에 따라 다름) |

**아이콘 색상으로 상태 확인**:
- 🟢 초록 = 작업 중 (코인 적립 중)
- 🟡 노랑 = 대기 중 (작업 대기)
- 🔴 빨강 = 일시정지 (게임/사용자 활동 감지)
- ⚫ 회색 = 미로그인 / 오프라인

### 로그인 + KYC

화랑 Grid Agent를 클릭하면 창이 뜹니다.

1. **로그인** — hwarang.ai 계정으로 로그인 (없으면 **회원가입** 클릭, 이메일+비밀번호로 1분이면 됩니다)
2. **KYC (본인 인증)** — 코인을 **출금**하려면 본인 인증이 필요합니다 (작업/적립은 본인 인증 없이도 가능). 신분증 사진 + 셀카 1장으로 보통 1~2일 안에 승인됩니다.
3. **지갑 연결** — HWR 코인을 받을 지갑 주소를 입력합니다. (없으면 **앱 내 지갑 자동 생성** 옵션 선택)

> *(스크린샷 자리)* — 로그인 화면 / KYC 안내 / 지갑 연결

## 3. GPU 확인

### 어떤 GPU가 지원되나요?

| 등급 | 모델 예시 | 시간당 예상 보상 |
|---|---|---|
| **하급** | GTX 1060, RTX 2060, AMD RX 580 | ~10 HWR/시간 |
| **중급** | RTX 3060, RTX 3070, RTX 4060 | ~30~50 HWR/시간 |
| **상급** | RTX 3080, RTX 4070, RTX 4080 | ~80~150 HWR/시간 |
| **프로** | RTX 4090, A6000 | ~200~400 HWR/시간 |
| **데이터센터** | A100, H100 | ~1000+ HWR/시간 |
| **Apple** | M1/M2/M3 (Pro/Max/Ultra 우대) | ~20~80 HWR/시간 |

> 보상은 **수요/공급/네트워크 활용도**에 따라 변동됩니다. 위 수치는 평균 추정치입니다.

### NVIDIA 드라이버 확인 (Windows / Linux)

#### Windows
1. **시작 → "장치 관리자" 검색 → 실행**
2. **디스플레이 어댑터** 펼치기 → NVIDIA 모델명 확인
3. 드라이버 없으면 https://www.nvidia.com/Download/index.aspx 에서 다운로드

#### Linux (Ubuntu)
```bash
nvidia-smi   # 출력되면 OK
# 안 되면:
sudo ubuntu-drivers autoinstall
sudo reboot
```

#### macOS (Apple Silicon)
- M1/M2/M3 자동 지원, 드라이버 설치 불필요
- Intel Mac은 GPU 작업 지원이 제한적

### CUDA / 드라이버 버전

화랑 에이전트가 **자동 감지**합니다. 부족하면 알림으로 안내합니다.

## 4. 프리셋 선택

화랑 네트워크는 다양한 분야의 AI 작업이 있습니다. **어떤 작업에 내 GPU를 빌려줄지** 선택할 수 있습니다.

| 프리셋 | 설명 | 예상 모델 | 적합한 GPU |
|---|---|---|---|
| **일반** | 가장 무난 (분야 무관) | Qwen, Llama 등 | 모든 GPU |
| **법률** | 한국어 법률 문서 처리 | EXAONE 기반 | 8GB VRAM 이상 |
| **의료** | 의료 텍스트/영상 분석 | 의료 특화 모델 | 12GB VRAM 이상 |
| **세무** | 세무 신고/회계 자동화 | 세무 특화 LoRA | 8GB VRAM 이상 |
| **코딩** | 코드 생성/리뷰 | DeepSeek, Qwen-Coder | 16GB VRAM 이상 |
| **이미지** | 이미지 생성 (SD/Flux) | Stable Diffusion | 8GB VRAM 이상 |

> 프리셋이 헷갈리시면 **일반(All-Round)** 으로 두세요. 가장 안정적입니다.

## 5. 돈 벌기 (HWR 코인)

### HWR 코인이란?

화랑 네트워크의 **자체 보상 토큰**입니다.

- GPU 작업 1건당 자동으로 적립
- 거래소 상장 시 한국 원화/USDT 로 환전 가능
- 화랑 생태계 내 서비스 결제에도 사용

### 예상 수익 표

> 24시간 풀가동 + 평균 부하 60% 기준 (실제는 수요에 따라 다름)

| GPU | 시간당 (HWR) | 하루 (HWR) | 한 달 (HWR) | 한 달 예상 (KRW)* |
|---|---:|---:|---:|---:|
| RTX 3060 | 30 | 720 | 21,600 | 2~3만원 |
| RTX 3080 | 100 | 2,400 | 72,000 | 7~10만원 |
| RTX 4090 | 300 | 7,200 | 216,000 | 22~30만원 |
| A100 | 1,500 | 36,000 | 1,080,000 | 100~150만원 |

> \* 1 HWR = 1.0~1.5 KRW 가정 (실제 시장가 변동)
>
> \* 전기세는 **별도** 입니다 (RTX 4090 풀가동 시 한 달 전기세 ~3~5만원)

### 정산 (출금) 방법

1. **트레이 아이콘 → 대시보드 열기**
2. **"내 자산"** 탭에서 누적 HWR 확인
3. **"출금"** 버튼 → 지갑 주소 + 수량 입력
4. KYC 완료된 계정만 출금 가능
5. 일반적으로 **5분~1시간 내** 입금 완료

### 정산 주기 (네트워크 결정)
- **즉시 출금**: 100 HWR 이상
- **자동 정산**: 매주 일요일 자정 (옵션 설정 시)

## 6. 자주 묻는 질문

### Q1. 게임할 때도 작동하나요?

**아니요. 자동으로 일시정지됩니다.**

화랑 에이전트는 다음을 감지하면 **즉시 작업을 중단**합니다:
- 풀스크린 게임 실행
- GPU 사용률 > 70% (다른 프로그램이 사용 중)
- 사용자 마우스/키보드 활동 (대기모드 옵션 켰을 때)

게임 종료 후 5분간 idle 상태가 지속되면 자동으로 재개합니다.

### Q2. 전기세는 얼마나 드나요?

**GPU 모델별 한 달 풀가동 기준:**

| GPU | 와트 (W) | 24시간 가동 시 한 달 전기세* |
|---|---:|---:|
| GTX 1660 | 120W | ~1만원 |
| RTX 3060 | 170W | ~1.5만원 |
| RTX 3080 | 320W | ~3만원 |
| RTX 4090 | 450W | ~4~5만원 |

> \* 한국 가정용 누진세 1단계 (kWh당 120원) 기준
>
> **수익이 전기세를 충분히 상회하지 않으면 알림**으로 안내드립니다.

### Q3. 안전한가요?

**네. 안전합니다.**

- 모든 작업은 **샌드박스(격리 환경)** 에서 실행
- 외부에서 여러분 컴퓨터의 **파일/네트워크에 접근 불가**
- 개인 정보, 비밀번호 등은 **절대 수집하지 않습니다**
- 오픈소스로 공개되어 누구나 코드 검증 가능

### Q4. 인터넷은 얼마나 사용하나요?

**평균 한 달 5~30GB 정도** (작업 종류에 따라 다름)

대부분 **다운로드** (모델 가중치)이고, 결과 업로드는 작습니다. 무제한 요금제면 부담 없습니다.

### Q5. 노트북에서도 돼요?

**됩니다. 그러나 권장은 안 합니다.**

- 노트북 GPU는 발열로 수명이 짧아질 수 있음
- 데스크탑 PC 또는 워크스테이션 권장
- 노트북은 **충전 중 + 쿨링 패드** 사용 시에만 켜는 것을 추천

### Q6. 한 PC에 GPU가 여러 개여도 되나요?

**네. 자동으로 모든 GPU를 활용합니다.**

대시보드에서 GPU별로 켜고 끄거나 우선순위를 설정할 수 있습니다.

### Q7. 모바일은 안 되나요?

**현재는 데스크탑(Windows/Mac/Linux)만 지원합니다.**

모바일 기기는 GPU 성능과 발열 문제로 효율이 낮아 지원 계획이 없습니다.

## 7. 문제 해결

### "트레이 아이콘이 안 보여요"

#### Windows
- 트레이 영역 **^** 클릭 → 화랑 아이콘 찾기
- **시작 메뉴 → "Hwarang Grid Agent" 검색** → 실행
- **작업 표시줄 우클릭 → 작업 표시줄 설정 → 시스템 트레이 아이콘** → 화랑 켜기

#### macOS
- 메뉴바에 공간이 부족하면 숨겨질 수 있음 → **Bartender** 같은 도구로 정리
- **Activity Monitor**(활성 상태 보기)에서 "Hwarang Grid Agent" 프로세스 확인

#### Linux
- KDE/XFCE는 자동 표시. GNOME은 **AppIndicator** 확장 필요:
  ```bash
  sudo apt install gnome-shell-extension-appindicator
  ```

### "보상이 안 들어와요"

체크 순서:
1. **트레이 아이콘이 초록색**인지? (회색이면 미로그인)
2. **인터넷 연결** 확인 (브라우저로 hwarang.ai 접속)
3. **KYC 완료** 했나요? (출금 시 필요)
4. **최소 출금 금액(100 HWR)** 이상인가요?
5. 그래도 안 되면: 트레이 → **로그 보내기** → 자동으로 운영팀에 전송됨

### "GPU가 인식 안 돼요"

#### Windows
1. **장치 관리자**에서 GPU 모델명 확인 (느낌표 / X 표시 있으면 드라이버 문제)
2. **GeForce Experience** 또는 NVIDIA 사이트에서 최신 드라이버 설치
3. PC 재부팅
4. 그래도 안 되면 트레이 → **GPU 재감지** 클릭

#### Linux
```bash
nvidia-smi   # 출력 확인
# 안 나오면:
sudo apt install nvidia-driver-535   # 또는 최신 버전
sudo reboot
```

### "PC가 너무 느려져요"

설정에서 **GPU 사용률 한도**를 낮추세요:
- 트레이 → **설정 → 성능 → 최대 GPU 사용률**
- **80% → 50%** 로 낮추면 게임/작업 영향 거의 없음

### 로그 보내기 (운영팀 도움 요청)

트레이 아이콘 → **도움말 → 로그 보내기** 클릭
- 자동으로 `~/.hwarang/logs/*.log` 가 안전하게 운영팀에 전송됩니다
- 개인 정보는 **자동으로 제거**됩니다

## 8. 제거하기

### macOS

```bash
# 자동 제거 스크립트 (권장)
curl -sL https://hwarang.ai/uninstall/mac.sh | bash
```

또는 수동:
1. **Finder → 응용 프로그램** → **Hwarang Grid Agent.app** → **휴지통으로 이동**
2. (선택) 사용자 데이터 삭제: 터미널에서 `rm -rf ~/.hwarang`

### Windows

1. **설정 → 앱 → 설치된 앱**에서 **Hwarang Grid Agent** 찾기
2. **제거** 클릭
3. (선택) 사용자 데이터 삭제: PowerShell에서 `Remove-Item -Recurse "$env:USERPROFILE\.hwarang"`

### Linux

```bash
# .deb 로 설치한 경우
sudo apt remove hwarang-grid-agent

# AppImage 로 설치한 경우
rm ~/Hwarang-Grid-Agent-*.AppImage

# 사용자 데이터
rm -rf ~/.hwarang
```
